# 📦 My Script Utility Folder

This folder contains automation scripts for trading automation, ngrok tunnel, and WhatsApp bot.

## 📄 Files:

1. **startbot.bat**  
   → Runs the Venom bot using `npm run start-bot`

2. **startbot.vbs**  
   → Launches startbot.bat in a hidden console window.

3. **startAll.js**  
   → Starts ngrok, updates webhook URLs, triggers Render deploy, and starts Venom bot.

4. **stopNgrok.bat**  
   → Terminates any running ngrok session.

## ⚙️ Usage:

- To start everything → `double-click startbot.bat` or run `npm run start`  
- To stop ngrok → `double-click stopNgrok.bat`  
- To edit webhook configs → modify `.env` file.
